# 🎄 豪华互动粒子圣诞树
## ✨ 项目介绍
这是一棵以 “记忆收藏” 为核心的3D 粒子互动圣诞树，颜值与温度兼备：

🎨 视觉风格：以深邃黑为底色，搭配鎏金粒子 + 圣诞红装饰，自带电影级辉光效果，夜晚打开直接拉满节日氛围感～

🧩 构成元素：由鎏金粒子、圣诞小装饰（圆球、糖果等）和照片云组成；粒子聚合时是饱满的圣诞树形状，顶部会亮起 “MERRY CHRISTMAS” 字样。

🖼️ 照片功能：点击 “ADD MEMORIES” 按钮，即可上传你的照片（圣诞合照、生活碎片都可以），这些照片会藏在粒子中，成为圣诞树的专属 “记忆碎片”~
## 🎮 交互逻辑
这棵树有三种核心状态，切换起来超有仪式感：

🌲 圣诞树合拢态：所有粒子、装饰、照片收拢聚合，形成完整的圣诞树造型，顶部显示 “MERRY CHRISTMAS”—— 经典圣诞感直接拉满！

✨ 粒子散开态：触发互动后，所有元素会缓慢漂浮散开，像把圣诞树 “拆成了星空”，藏在里面的照片也会跟着粒子一起悬浮～

 🖼️ 照片聚焦态：在散开态下选中任意照片，照片会放大展示，背景保持粒子漂浮的梦幻效果 —— 相当于从 “圣诞星空” 里捞出一段专属回忆～
## ✋ 手势控制说明
请先允许浏览器访问摄像头，在镜头前做以下手势即可操控：
| 手势       | 动作描述       | 功能                                   |
| :--------- | :------------- | :------------------------------------- |
| **✊ 握拳** | Fist           | 回到 圣诞树合拢态（粒子聚合为树）       |
| **🖐️ 打开五指** | Open Palm      | 切换到 粒子散开态（元素漂浮）           |
| **👋 手掌平移** | Hand Move      | 在散开态下，跟随手掌调整画面视角       |
| **👌 指尖捏合** | Pinch          | 选中并放大单张照片，进入照片聚焦态     |
## 🛠️ 如何使用
下载本仓库中的 .html 文件，用 Chrome 浏览器（兼容性最佳）打开该文件；
（或者直接打开网页版链接：https://mellow-crisp-3549fa.netlify.app/）

允许浏览器请求的 “摄像头权限”（用于手势识别）；

点击界面 “ADD MEMORIES” 按钮，上传你想收藏的照片；

按键盘 “H” 键可隐藏 / 显示控制项，获得沉浸式体验；

对着摄像头挥动双手，开启你的圣诞记忆之旅吧！
